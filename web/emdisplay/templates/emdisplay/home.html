    emdsiplay

    On these pages the system emdisplay can be configured.

    Configuration

    The basic settings of the system can be made there, such as the directory in which the incoming fax should be searched for or the host name of the SMTP server.

    Fax Parsing Templates

    With this page you can create and design regular expressions to analyze and parse incoming fax messages.

    Events

    This page lists the fax messages that have already been received. If the fax was detected as an alert, the detected detail fields are listed. In addition, the fax can be viewed as an image and how the OCR recognized the fax.

    Receiver

    There the persons can be managed with name, phone number, e-mail address and so on which can receive messages from system.

    Messaging Templates

    On the Messaging Templates page, templates can be managed and edited which are used to notify the individual receiver of the system.
